Bracelet Market Research

A private dashboard for studying demand and competition across crystal, gemstone, beaded, and chakra bracelets to help me plan my own Etsy shop with real signals instead of guesswork.

Capabilities

What it does

Reads public listings

Searches active public listings and public reviews — never edits anything. Read-only access to Etsy's Open API v3.

Scores the opportunity

Aggregates favorites, review counts, prices, and listing volume into a demand-vs-competition view for each niche.

Compares four niches

Places crystal, gemstone, beaded, and chakra bracelets side by side to surface the underserved gaps worth entering.

Read-only · Data transparency

This tool reads public data only. It does not modify listings, place orders, or share or resell Etsy data. All API usage complies with Etsy's Open API Terms of Use.
